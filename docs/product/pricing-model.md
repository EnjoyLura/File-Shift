# FileShift 积分定价模型

## 1. 积分体系设计

### 1.1 核心理念

- 积分是平台唯一的消费货币
- 低门槛体验：新用户赠送足够体验核心功能的积分
- 灵活消费：按需付费，不设月费/年费包
- 鼓励传播：邀请机制实现用户自增长

### 1.2 积分与人民币换算

- **基础汇率**：1元 ≈ 10积分
- 大额购买有折扣优惠

## 3. 积分消耗标准

### 3.1 定价原则

- 操作复杂度：简单操作低消耗，复杂操作高消耗
- 资源占用：CPU/内存/时间密集型操作消耗更高
- 市场对标：确保比竞品单次使用成本更低

### 3.2 文档格式转换

| 功能            | 积分/次 | 定价理由                   |
| --------------- | ------- | -------------------------- |
| PDF → Word      | 3       | 核心高频功能，保持低价引流 |
| Word → PDF      | 2       | 简单转换，低消耗           |
| PDF → Excel     | 5       | 表格识别复杂度高           |
| Excel → PDF     | 2       | 简单转换                   |
| PDF → PPT       | 5       | 结构化提取复杂             |
| PPT → PDF       | 2       | 简单转换                   |
| PDF → 图片      | 2       | 按文件计费(不按页)         |
| 图片 → PDF      | 2       | 简单合并                   |
| Markdown → PDF  | 2       | 轻量渲染                   |
| Markdown → HTML | 1       | 极简转换                   |

## 4. 用户积分使用场景模拟

### 4.1 新用户50积分能做什么

| 使用方式        | 消耗积分 | 剩余 |
| --------------- | -------- | ---- |
| 转换5张图片格式 | 5        | 45   |
| PDF转Word 2次   | 6        | 39   |
| 压缩10张图片    | 10       | 29   |
| PDF合并1次      | 2        | 27   |
| 视频格式转换1次 | 5        | 22   |
| OCR识别2次      | 10       | 12   |
| 图片裁剪5次     | 5        | 7    |

**结论**：50积分可完成约25-30次操作，足够深度体验产品价值。

### 4.2 轻度用户月消耗估算

- 月均使用10-20次 = 15-40积分
- 推荐购买：体验包(100积分/9.9元)，可用2-3个月

### 4.3 重度用户月消耗估算

- 月均使用50-100次 = 100-300积分
- 推荐购买：标准包(600积分/45元)，可用2-3个月

## 5. 运营策略

### 5.1 拉新策略

- 新用户赠送50积分（降低尝试门槛）
- 邀请奖励20积分（鼓励口碑传播）
- 首充优惠：首次充值额外赠送20%积分

### 5.2 促活策略

- 积分即将过期提醒（到期前7天邮件/站内通知）
- 节假日充值折扣活动
- 连续使用3天奖励5积分（未来版本）

### 5.3 转化策略

- 积分不足时明确提示"再充X元即可继续"
- 操作结果预览页展示效果，激发付费意愿
- 套餐页突出性价比对比

### 5.4 防薅策略

- 同一设备指纹只能注册1个账号获得赠送积分
- 邀请奖励上限：每人每月最多邀请50人
- 异常行为检测：短时间大量注册同IP封禁

## 6. 收入预估模型

### 假设条件

- 日活用户(DAU): 500人
- 付费转化率: 5%
- 月均ARPU(付费用户): 30元

### 月收入估算

- 付费用户数: 500 × 5% = 25人
- 月收入: 25 × 30 = **750元/月**

### 增长目标

| 阶段           | DAU       | 付费用户 | 月收入      |
| -------------- | --------- | -------- | ----------- |
| 初期(1-3月)    | 100-500   | 5-25     | 150-750元   |
| 增长期(4-6月)  | 500-2000  | 25-100   | 750-3000元  |
| 稳定期(7-12月) | 2000-5000 | 100-250  | 3000-7500元 |

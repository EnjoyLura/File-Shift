# FileShift 用户流程文档

## 1. 核心用户流程

### 1.1 总体流程概览

```mermaid
graph TB
    A[访问首页] --> B{是否登录}
    B -->|未登录| C[浏览功能列表]
    B -->|已登录| D[选择转换功能]
    C --> E[点击使用]
    E --> F[跳转登录/注册]
    F --> G[登录成功]
    G --> D
    D --> H[上传文件]
    H --> I[设置转换参数]
    I --> J{积分是否足够}
    J -->|不足| K[提示充值]
    J -->|足够| L[开始转换]
    L --> M[等待转换完成]
    M --> N[下载转换结果]
    N --> O[完成]
```

## 2. 注册登录流程

```mermaid
graph TB
    A[点击注册] --> B[输入邮箱]
    B --> C[输入密码]

    E --> F{验证通过}
    F -->|是| G[注册成功]
    F -->|否| H[提示错误重试]

graph TB

    C -->|否| E[提示错误]
    D --> F[签发JWT Token]
    F --> G[跳转来源页/首页]
```

## 3. 文件转换流程

### 3.1 单文件转换

```mermaid
graph TB
    A[选择转换类型] --> B[点击上传区域/拖拽文件]
    B --> C{文件校验}
    C -->|格式不符| D[提示文件格式错误]
    C -->|超出大小| E[提示文件过大]
    C -->|通过| F[上传文件到服务器]
    F --> G[显示上传进度]
    G --> H[上传完成]
    H --> I[选择输出格式/参数]
    I --> J[点击开始转换]
    J --> K{检查积分}
    K -->|不足| L[弹出积分不足提示]
    K -->|足够| M[扣减积分]
    M --> N[创建转换任务]
    N --> O[进入任务队列]
    O --> P[显示转换进度]
    P --> Q[转换完成]
    Q --> R[显示下载按钮]
    R --> S[点击下载]
    S --> T[文件下载到本地]
```

### 3.2 批量文件转换

```mermaid
graph TB
    A[选择批量转换] --> B[上传多个文件 ≤20个]
    B --> C[显示文件列表]
    C --> D[统一设置输出格式]
    D --> E[显示所需积分总数]
    E --> F{积分足够}
    F -->|否| G[提示充值]
    F -->|是| H[批量扣减积分]
    H --> I[创建批量任务]
    I --> J[逐个转换+进度显示]
    J --> K[全部完成]
    K --> L[打包下载ZIP / 逐个下载]
```

### 3.3 文件转换状态机

```
待上传 → 上传中 → 已上传 → 排队中 → 转换中 → 转换成功 → 已下载
                                              ↓
                                         转换失败 → 可重试(不扣积分)
```

## 4. 积分相关流程

### 4.1 积分获取途径

| 途径       | 积分数量 | 条件               |
| ---------- | -------- | ------------------ |
| 新用户注册 | +50      | 一次性             |
| 邀请新用户 | +20 / 人 | 被邀请人完成注册   |
| 购买积分包 | 按套餐   | 支付成功后即时到账 |

### 4.2 积分充值流程

```mermaid
graph TB
    A[点击充值] --> B[选择积分套餐]
    B --> C[选择支付方式]
    C --> D{微信/支付宝}
    D -->|微信| E[生成微信支付二维码]
    D -->|支付宝| F[跳转支付宝支付页]
    E --> G[支付完成回调]
    F --> G
    G --> H[验证支付结果]
    H -->|成功| I[积分到账]
    H -->|失败| J[支付失败提示]
    I --> K[更新积分余额显示]
```

### 4.3 邀请流程

```mermaid
graph TB
    A[进入邀请页面] --> B[获取专属邀请链接/邀请码]
    B --> C[分享给朋友]
    C --> D[朋友通过链接注册]
    D --> E[系统验证邀请关系]
    E --> F[邀请人+20积分]
    E --> G[被邀请人+50积分注册奖励]
```

## 5. 个人中心流程

### 5.1 用户中心功能

```mermaid
graph TB
    A[个人中心] --> B[基本信息]
    A --> C[积分管理]
    A --> D[历史记录]
    A --> E[收藏功能]
    A --> F[账号安全]

    B --> B1[修改昵称/头像]
    C --> C1[积分余额]
    C --> C2[积分流水]
    C --> C3[去充值]
    D --> D1[转换历史列表]
    D --> D2[重新下载未过期文件]
    E --> E1[收藏的工具]
    F --> F1[修改密码]
    F --> F2[绑定手机/微信]
```

### 5.2 历史记录

| 字段     | 说明                       |
| -------- | -------------------------- |
| 文件名   | 原始文件名称               |
| 转换类型 | 如 PDF→Word                |
| 状态     | 成功/失败/进行中           |
| 消耗积分 | 本次操作消耗               |
| 创建时间 | 任务创建时间               |
| 操作     | 重新下载(24h内) / 再次转换 |

## 7. 异常流程处理

| 异常场景          | 处理方式                             |
| ----------------- | ------------------------------------ |
| 文件上传中断      | 显示上传失败，提供重试按钮           |
| 转换超时          | 标记为失败，不扣积分，提示用户重试   |
| 转换失败          | 不扣积分，显示失败原因，提供重试选项 |
| 积分不足          | 阻止操作，引导充值页面               |
| 文件已过期(超24h) | 提示文件已清理，建议重新转换         |
| 网络断开          | 前端检测后提示，恢复后自动重连       |
| 服务器维护        | 显示维护页面，预计恢复时间           |
| Token过期         | 自动刷新Token，刷新失败跳转登录      |

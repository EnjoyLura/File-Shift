# FileShift 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位

FileShift 是一款在线文件格式转换工具站，提供文档、图片、音视频格式转换，文件压缩，以及各类实用小工具。面向有文件处理需求的个人用户和轻量办公场景。

### 1.2 目标用户画像

| 用户群体        | 特征                     | 核心需求                        |
| --------------- | ------------------------ | ------------------------------- |
| 大学生/职场新人 | 经常处理论文、简历、作业 | PDF↔Word、图片压缩、PDF合并     |
| 自媒体创作者    | 频繁处理图片和视频素材   | 图片格式转换、视频压缩、GIF制作 |
| 小型企业/个体户 | 日常办公文件处理         | 文档互转、批量处理、OCR识别     |
| 普通网民        | 偶尔有格式转换需求       | 图片转换、视频格式转换          |

### 1.3 竞品分析

| 竞品             | 优势             | 劣势                     |
| ---------------- | ---------------- | ------------------------ |
| iLovePDF         | 品牌知名、功能全 | 国外服务慢、付费贵       |
| SmallPDF         | UI好、体验流畅   | 免费次数少、不支持音视频 |
| 格式工厂(客户端) | 免费、功能多     | 需下载安装、有广告       |
| 迅捷在线转换     | 中文友好         | 功能单一、价格高         |

**FileShift差异化**：积分制灵活付费 + 功能全覆盖(文档+图片+音视频) + 国内服务器速度快 + 响应式支持手机端

## 3. MVP 定义（最小可行产品）

### MVP 范围（优先交付的P0功能）

**核心转换**：

- PDF ↔ Word
- PDF → 图片 / 图片 → PDF
- PNG ↔ JPG ↔ WebP
- HEIC → JPG
- MP4 ↔ AVI
- MP3 ↔ WAV
- 视频提取音频

**核心工具**：

- 图片压缩
- 视频压缩
- PDF合并/拆分
- 图片裁剪

**基础功能**：

- 邮箱注册登录
- 积分系统（注册赠送、消耗扣减）
- 文件上传/下载
- 个人中心/历史记录

### MVP 不包含

- 微信/手机号登录（阶段5接入）
- 支付充值（阶段5接入）
- 邀请系统（阶段5接入）
- 管理后台（阶段5接入）
- SEO优化（阶段5接入）

## 4. 非功能需求

| 类别     | 要求                                                          |
| -------- | ------------------------------------------------------------- |
| 性能     | 图片转换 < 3s，文档转换 < 30s，音视频转换 < 5min              |
| 文件限制 | 免费用户单文件 ≤ 20MB，付费用户 ≤ 100MB                       |
| 并发     | 同一用户同时最多3个转换任务                                   |
| 可用性   | 服务可用率 > 99%                                              |
| 安全性   | 上传文件24h后自动删除，不存储用户文件内容                     |
| 兼容性   | Chrome/Firefox/Safari/Edge最新两个版本，iOS/Android手机浏览器 |
| 响应式   | 完整支持 PC(1920px) / 平板(768px) / 手机(375px)               |

## 5. 用户权限设计

| 用户类型               | 权限                                       |
| ---------------------- | ------------------------------------------ |
| 未登录用户             | 可浏览功能列表，不可使用转换功能           |
| 已注册免费用户         | 使用所有功能，受积分限制，单文件 ≤ 20MB    |
| 付费用户（购买过积分） | 使用所有功能，单文件 ≤ 100MB，转换优先队列 |
| 管理员                 | 后台管理、用户管理、数据统计               |
